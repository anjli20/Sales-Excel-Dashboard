# Excel Sales Dashboard

An interactive Excel dashboard for analyzing sales executive performance across multiple regions. The dashboard helps track total sales,
target achievement, and performance gaps using slicers, summary tables, and charts.

## Project Description

This project presents a sales performance dashboard built in Microsoft Excel. It allows users to view and compare sales data by region and 
sales executive. The dashboard includes key metrics such as total sales, target hit percentage, and away-from-target percentage.

## Features

- Interactive region-based filtering using slicers
- Sales executive-wise performance summary
- Total sales analysis
- Target hit percentage tracking
- Away-from-target percentage analysis
- Bar chart, pie chart, and line chart visualizations
- Clean and user-friendly dashboard layout

## Dashboard Insights

The dashboard provides a quick view of:

- Top-performing sales executives
- Region-wise sales performance
- Sales executives meeting or missing targets
- Percentage gap from assigned targets
- Overall sales trends and comparisons
